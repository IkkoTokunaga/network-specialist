# VPN（Virtual Private Network）
インターネット上に暗号化した仮想的な専用線を作成し、色々な拠点で色々なユーザを接続する機能
## 拠点間VPN
拠点同士を接続するVPN。昔は拠点間に専用線を敷設して接続していたが多額の資金が必要であった
## リモートアクセスVPN
VPNを接続できる専用のソフトウェアを通じて社内のネットワークにアクセスし、社内と同じように作業ができる。
「IPsec VPN」と「SSL-VPN」があったがIPsec VPNはNAPTと相性が悪かったため、SSL-VPNが主流となっている
| 観点       | IPsec | SSL VPN |
| -------- | ----- | ------- |
| 暗号化層     | L3    | L4〜L7   |
| ネットワーク秘匿 | ◎     | △       |
| 相互認証     | ◎     | △       |
| 攻撃面      | 小     | 大       |
| 管理容易性    | △     | ◎       |
IPsecはL3でTCPヘッダーを丸ごと暗号化するので秘匿性が高い
ただポート番号などの情報も暗号化するため、NATやFW越えには運用面で工夫が必要
## 脱VPNについて
これは拠点間VPNでつないでいる閉域網をオープンなSaasやアプリケーションを使用して変更していこうといった流れ
リモートワークをやめて出社させようとか、VPNを無理に廃止しようなどといった意図ではない
（そういった意図で使用されることも多い）
