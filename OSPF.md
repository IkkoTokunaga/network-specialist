# OSPF （Open Shotest Path First）
リンクステート型ルーティングプロトコル
メトリック：コスト
更新間隔：更新があった時
中規模～大規模
ポート番号89

## リンクステートデータベース（LSDB）
各ルータが
リンクの状態、帯域
IPアドレス
ネットワーク
など色々な状態を交換し、集めて作成されるデータベース

## Helloパケット
情報交換の相手に小さなパケットを送信して死活監視を行っている

## ECMP（Equal Cost Multi Path）
複数の経路が存在し、コストが全く同数の場合に複数経路を使用して通信を行って負荷分散を行う仕組みのこと
障害体制だけでなく、帯域の拡張も行えるのが利点
## OSPF流れ
1. Helloパケット送信
    OSPF使えるルータを探す
2．ネイバー確立
    お互いOSPFを確認
3．LSA交換（Link State Advertisement）
    自分の接続情報を交換
4．トポロジDB作成
    ネットワーク全体の地図を作る
5．SPF計算
    最短経路ツリー作成
6．ルーティングテーブル作成
### LSA（リンクステート広告）
例
```
RouterA

接続先
B const 10
D const 5
```
このような情報を洪水的に（フラッディング）配布
