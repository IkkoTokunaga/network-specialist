# GARP
+ 競合IPアドレスの検知。
    + OSがIPアドレスを設定する際に一旦ネットワーク内にブロードキャストして競合がないか確認を行い、該当あればエラーを吐きだす
+ 隣接機器のテーブル更新
    + 交換した機器は交換前の機器のIPアドレスを設定できるが、MACアドレスは変更となる
    + 交換した機器を起動した際にブロードキャストして、ARPテーブルを更新する
    + 機器がGARPしない場合は手動でARPテーブルをクリアする必要がある
    + 仮想マシンがネットワークを移動（ライブマイグレーション）を行った際もGARPをキャストして、スイッチのMACアドレステーブルを更新する
