# 2進数
0ち1だけで表現される数字のこと
## 計算方法
例：100
### 基準値を使用する方法
128 64  32  16  8   4   2   1   といった基準値を用意し、元の数より多いかを判定する

0=false 1=true で考えると分かりやすい
128 < 100 = 0
64 < 100 = 1 ※1だったら元の数から基準値を減算する
32 < (100-64=36) = 1
16 < (36-32=4) = 0
8 < 4 = 0
4 < 4 = 1
2 < (2-4=-2) = 0
1 < -2 = 0 

01100100

### 割り算から
0になるまで2で割って余りの数値をしたから並べる
※0になるまで！
100%2=0
50%2=0
25%2=1
12%2=0
6%2=0
3%2=1
1%2=1
