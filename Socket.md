# Socket
プログラム同士がネットワーク越しに通信するためのインターフェース

サーバ：ポートを開けて「待ち受け」
クライアント：そのポートに「接続」

## Socketの処理
| 処理          | 意味          |
| ----------- | ----------- |
| `socket()`  | 通信口を作る      |
| `bind()`    | IP:Port を確保 |
| `listen()`  | 接続待ち        |
| `accept()`  | 接続を受け取る     |
| `recv()`    | データ受信       |
| `sendall()` | データ送信       |
