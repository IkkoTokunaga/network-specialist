# BGP（Border Gateway Protocol）
ASとASをつなぐプロトコル（EGP）の一種
## AS番号
2バイトと4バイト AS番号があって、現在は4バイトが主流
AS識別用の一意な番号
| 種類           | 範囲                |
| ------------ | ----------------- |
| **2バイトAS番号** | 1 ～ 65,535        |
| **4バイトAS番号** | 1 ～ 4,294,967,295 |

グローバル用とプライベート用が存在する。
プライベート用はIPアドレスのように異なるAS間ではかぶってもOK
| AS番号長 | プライベート範囲                |
| ----- | ----------------------- |
| 2バイト  | 64512 ～ 65534           |
| 4バイト  | 4200000000 ～ 4294967294 |
## ルーティングアルゴリズム
パスベクタ型プロトコル
パス（経路）とベクタ（方向）に基づいてルートを計算
## ベストパス選択アルゴリズム
どの経路が最適なパスかを判断するか、そのルール
