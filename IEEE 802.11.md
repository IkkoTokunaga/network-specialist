# IEEE 802.11 無線LAN規格
| 規格       | 通称      | 最大速度    | 周波数      |
| -------- | ------- | ------- | -------- |
| 802.11b  | -       | 11Mbps  | 2.4GHz   |
| 802.11a  | -       | 54Mbps  | 5GHz     |
| 802.11g  | -       | 54Mbps  | 2.4GHz   |
| 802.11n  | Wi-Fi 4 | 600Mbps | 両方       |
| 802.11ac | Wi-Fi 5 | 数Gbps   | 5GHz     |
| 802.11ax | Wi-Fi 6 | さらに高速   | 2.4/5GHz |
## GHz
Hzは1秒間の振動回数
2.4GHz = 1秒間に24億回の振動

| 特性     | 低い周波数（2.4GHz） | 高い周波数（5GHz） |
| ------ | ------------- | ----------- |
| 壁に強い   | 強い            | 弱い          |
| 遠くまで届く | 届きやすい         | 届きにくい       |
| 速度     | やや遅い          | 高速化しやすい     |
| 混雑     | しやすい          | 比較的少ない      |
