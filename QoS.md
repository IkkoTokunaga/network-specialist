# Qos（Quality of Service）
ネットワークで通信の優先順位や品質を制御する仕組み
| 通信         | 特徴       |
| ---------- | -------- |
| IP電話       | 遅延に弱い    |
| 動画         | 帯域を大量に使う |
| ファイルダウンロード | 遅れても問題ない |
上記の場合はIP電話を最優先で流し、次に動画、最後にファイルDLといった流れになる
| 用語       | 意味         |
| -------- | ---------- |
| DSCP     | IPヘッダの優先度  |
| DiffServ | QoS方式      |
| IntServ  | 予約型QoS     |
| WFQ      | 重み付きキュー    |
| ポリシング    | 帯域制限       |
| シェーピング   | 通信をなだらかにする |
